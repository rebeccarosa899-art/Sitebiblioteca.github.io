# Site da Biblioteca Machado de Assis

Este projeto foi desenvolvido como atividade da disciplina de Desenvolvimento Web.

## Tecnologias utilizadas
- HTML5
- CSS3

## Páginas do projeto
- index.html
- sobre.html
- acervo.html
- serviços.html
- eventos.html
- contato.html

## Objetivo
Criar um site institucional para uma biblioteca, apresentando informações sobre seu acervo, serviços, eventos e formas de contato.

Desenvolvido por Rebecca Rosa Bragança Barbosa.
